🧠 Тема проекта: Social Battery Tracker

Social Battery Tracker — это веб-приложение для отслеживания и анализа уровня социальной энергии пользователя в течение дня.

Современный человек часто сталкивается с эмоциональным и социальным выгоранием, но не всегда понимает, какие именно действия или события влияют на его состояние. Данный проект направлен на решение этой проблемы через удобный инструмент самонаблюдения и аналитики.

🎯 Цель проекта

Создать приложение, которое позволяет:

отслеживать текущий уровень энергии (социальной “батарейки”)

фиксировать события, влияющие на состояние пользователя

анализировать, какие действия приводят к утомлению или восстановлению

⚙️ Основной функционал

Пользователь может:

устанавливать и изменять уровень энергии (от 0 до 100)

добавлять события с указанием их влияния (положительного или отрицательного)

просматривать историю изменений

анализировать статистику по своим действиям

📊 Особенности

автоматическое изменение уровня энергии при добавлении событий

простая аналитика (что “разряжает” и что “заряжает”)

визуализация данных (графики, динамика энергии)

минималистичный и удобный интерфейс

🛠️ Используемые технологии

Vue 3 (frontend)

Vite (сборка проекта)

Pinia (управление состоянием)

LocalStorage (хранение данных)

💡 Практическая значимость

Приложение может быть полезно:

людям, испытывающим социальную усталость

разработчикам и специалистам с высокой когнитивной нагрузкой

всем, кто хочет лучше понимать своё состояние и управлять энергией
